# MBTI 沉浸自习室

基于 16 种 MBTI 人格类型的沉浸式在线自习空间。每种人格对应独特的场景主题、视觉效果和环境音。

## 功能

- **16 种 MBTI 场景** — 每种人格有专属背景、粒子效果和环境音
- **番茄钟计时器** — 专注/短休/长休模式，支持自定义时长
- **声音系统** — 5 通道混音器（雨声/炉火/风声/咖啡/森林）+ 生成式背景音乐
- **粒子特效** — 星空、雨滴、萤火虫、火星、花瓣等 8 种效果
- **学习工具** — 待办清单、学习日记、成就系统
- **护眼提醒** — 每 20 分钟提示远眺
- **专注模式** — 隐藏所有 UI，沉浸式体验
- **日间/夜间模式** — 全局主题切换
- **键盘快捷键** — Space(开始暂停) / R(重置) / F(专注) / Esc(退出)
- **响应式设计** — 支持桌面和移动端

## 技术栈

- 纯前端单文件应用（零依赖）
- 原生 JavaScript
- CSS 变量 + 暗色主题
- Web Audio API（环境音 + 生成式音乐）
- Canvas 2D（粒子系统）
- localStorage 持久化
