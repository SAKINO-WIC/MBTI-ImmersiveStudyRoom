# MBTI 沉浸自习室

基于 16 种 MBTI 人格类型的沉浸式在线自习空间。每种人格对应独特的场景主题、视觉效果和环境音。

## 功能

- **17 种场景** — 16 种 MBTI + 隐藏彩蛋「阿拉斯加海湾」
- **场景专属 BGM** — 每个场景自动切换背景音乐
- **番茄钟计时器** — 专注/短休/长休模式，支持自定义时长
- **5 通道音效混音器** — 雨声/炉火/风声/咖啡/森林，场景预设
- **粒子特效** — 星空、雨滴、萤火虫、火星、花瓣、海浪等效果
- **学习工具** — 待办清单、学习日记、成就系统
- **护眼提醒** — 每 20 分钟提示远眺
- **专注模式** — 隐藏所有 UI，沉浸式体验
- **日间/夜间模式** — 全局主题切换
- **键盘快捷键** — Space(开始暂停) / R(重置) / F(专注) / Esc(退出)
- **响应式设计** — 桌面、平板、手机自适应

## 隐藏彩蛋

在选择页快速点击标题 **「找到你的学习氛围」5 次**，进入阿拉斯加海湾场景。

## 技术栈

- 纯前端单文件应用（零依赖）
- 原生 JavaScript + CSS 变量
- Web Audio API（环境音混音）
- Canvas 2D（粒子系统）
- HTML5 Audio（场景 BGM）
- localStorage 持久化
