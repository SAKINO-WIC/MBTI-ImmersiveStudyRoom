### 5️⃣ **LifeSystem - 生命指数系统** ⭐⭐⭐⭐⭐

#### 核心概念
用户的专注行为是"养分"，场景中的元素会随着累计学习数据而成长、变化。每个场景从"静态背景"进化为"活的生态系统"。

#### 四维成长指标

| 指标 | 计算方式 | 效果范围 | 实现细节 |
|------|---------|---------|---------|
| **Level (等级)** | `floor(totalSessions/10) + 1` (max 10) | 光球大小 | `baseScale = 0.6 + level × 0.04` |
| **Vitality (今日活力)** | `todaySessions / dailyGoal` (0~1) | 光球亮度 | `opacity = 0.05 + vitality × 0.1` |
| **Maturity (成熟度)** | `totalSessions / 100` (0~1) | 粒子密度 | `count = base × (0.5 + maturity × 0.8)` |
| **StreakPower (连续力量)** | `streak / 30` (0~1) | 颜色温暖度 | `warmth = streakPower × 0.15` |

#### 场景变化效果

**Level 1 (新手)**:
- 光球较小 (baseScale = 0.64)
- 粒子稀疏 (50% 基础)
- 颜色冷淡

**Level 5 (熟练)**:
- 光球中等 (baseScale = 0.80)
- 粒子适中 (70% 基础)
- 颜色微暖

**Level 10 (大师)**:
- 光球巨大 (baseScale = 1.0)
- 粒子密集 (130% 基础)
- 颜色温暖

```javascript
baseCounts = {

```

**密度计算**:
```javascript
targetCount = base × (0.5 + maturity × 0.8)
```

| 成熟度 | 粒子倍率 | 实际粒子数 (stars) |
|--------|---------|-------------------|

#### 增量更新机制

当目标数量 > 当前数量时，每次更新周期添加最多 5 个粒子：
```javascript
var toAdd = Math.min(target - current, 5);
```

**优势**:
- ✅ 平滑过渡，无视觉跳跃
- ✅ 无性能冲击
- ✅ 粒子自然分布 (使用相同的随机生成逻辑)

#### 粒子速度缓存

**关键设计**:
```javascript
sleep: function() {
  for (var i = 0; i < fx.p.length; i++) {
    if (fx.p[i].vx) fx.p[i]._vx = fx.p[i].vx;  // 缓存原始速度
    if (fx.p[i].vy) fx.p[i]._vy = fx.p[i].vy;
    if (fx.p[i].vx) fx.p[i].vx *= 0.3;           // 减速到 30%
    if (fx.p[i].vy) fx.p[i].vy *= 0.3;
  }
}
wake: function() {
  for (var i = 0; i < fx.p.length; i++) {
    if (fx.p[i]._vx !== undefined) fx.p[i].vx = fx.p[i]._vx;  // 恢复原始速度
    if (fx.p[i]._vy !== undefined) fx.p[i].vy = fx.p[i]._vy;
  }
}
```

**优势**:
- ✅ 保存原始速度，恢复时完全还原
- ✅ 粒子不会"跳"或位置异常
- ✅ 平滑过渡 (CSS transition 2s)

#### 恢复机制

**触发条件**: 任何用户交互事件

**效果**:
- 立即清除空闲定时器
- 恢复粒子原始速度
- CSS 过渡动画平滑恢复亮度/饱和度
- 用户无感知 (无缝切换)
